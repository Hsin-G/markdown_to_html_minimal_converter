Markdown to HTML Converter (Go)

A simple Go program that converts a Markdown (.md or .mk) file into an HTML file using Goldmark.

Features

  • Converts .md and .mk files to HTML
  • Simple CLI usage

Installation

git clone https://github.com/yourusername/markdown-to-html.git
cd markdown-to-html
go mod tidy

Usage

go run main.go ./example/example.md

or

go build main.go
./<executable> ./example/example.md

Example

Input (example.md)

# Hello
This is **Markdown**.

Output (output.html)

<h1>Hello</h1>
<p>This is <strong>Markdown</strong>.</p>
